Документация campfire.py
==========================

Описание
----------

Модуль был создан для разработки ботов и скриптов для соцсети Campfire.
Он содержит в себе инструкции, из которых строятся запросы на сервер Campfire, а из ответа от сервера, при возможности, строятся объекты, с которыми можно совершать различные запросы на сервер.

Установка
-----------

Пока что, модуль возможно скачать только с `Github <https://github.com/Camper-CoolDie/campfire.py/tree/main/campfire>`_, либо его официальные версии в `Campfire <http://campfiresayzen.net/r/r.php?a=rubric-474>`_.
